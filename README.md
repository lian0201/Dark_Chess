# Dark Chess (暗棋) - C 語言與 Raylib 實作

這是一個使用 C 語言與 [Raylib](https://www.raylib.com/) 繪圖函式庫開發的經典「象棋暗棋」遊戲。目前已實作基礎的棋盤渲染、中文字型顯示、洗牌邏輯，以及基礎的玩家與電腦輪流翻牌功能。

## 🌟 遊戲特色

* **經典 4x8 棋盤配置**：還原傳統暗棋的 32 顆棋子與隨機洗牌機制。
* **完美中文顯示**：透過載入 `.ttf` 字型檔，解決終端機與繪圖視窗的中文亂碼問題。
* **即時資訊面板**：畫面上方會即時顯示「遊戲時間」、「目前回合（玩家／電腦）」以及「總手數」。
* **基礎 PVE 模式**：玩家點擊滑鼠翻開棋子後，會切換至電腦回合（電腦目前會自動隨機翻開一顆未知的棋子）。

## 📁 專案結構

確保你的資料夾層級如下，才能正確編譯與執行：

```text
.
├── include/           # Raylib 的標頭檔 (如 raylib.h)
├── lib/               # Raylib 的靜態函式庫 (如 libraylib.a)
├── Dark_Chess.c       # 遊戲主程式碼
├── chinese.ttf        # 遊戲使用的中文黑體字型檔
└── README.md          # 專案說明文件
```

## 🚀 快速啟動 (關鍵三步)

要執行這個遊戲，請跟著以下三個最關鍵的步驟操作：

### 1️⃣ 使用什麼開啟？
請使用 **Visual Studio Code (VS Code)**，透過「檔案 > 開啟資料夾」選擇本專案的根目錄。然後在左側檔案總管中點擊打開 **`Dark_Chess.c`** 檔案。

### 2️⃣ 要開啟哪個視窗？
在 VS Code 上方的選單列選擇「終端機 (Terminal) > 新增終端機」，在畫面下方打開終端機視窗，並確認目前的路徑是在這個專案資料夾內。

### 3️⃣ 要輸入什麼指令？
在終端機中，**依序**複製貼上並執行以下兩行指令：

**第一行（編譯指令，請一字不漏複製）：**
```bash
gcc Dark_Chess.c -o DarkChess.exe -I include/ -L lib/ -lraylib -lopengl32 -lgdi32 -lwinmm -fexec-charset=utf-8 -finput-charset=utf-8
```

**第二行（執行遊戲指令）：**
```bash
.\DarkChess.exe
```

## 🎮 遊玩操作

* **滑鼠左鍵**：點擊畫面上尚未翻開的灰色暗棋，即可將其翻開並顯示出對應的棋子種類與顏色（紅方或黑方）。
* 玩家操作完畢後，需等待約半秒鐘，電腦會自動執行它的回合。
