# Dark Chess (暗棋) - C 語言與 Raylib 實作

這是一個使用 C 語言與 [Raylib](https://www.raylib.com/) 繪圖函式庫開發的經典「象棋暗棋」遊戲。目前已實作基礎的棋盤渲染、中文字型顯示、洗牌邏輯，以及基礎的玩家與電腦輪流翻牌功能。

## 🌟 遊戲特色

* **經典 4x8 棋盤配置**：還原傳統暗棋的 32 顆棋子與隨機洗牌機制。
* **完美中文顯示**：透過載入 `.ttf` 字型檔，解決終端機與繪圖視窗的中文亂碼問題。
* **即時資訊面板**：畫面上方會即時顯示「遊戲時間」、「目前回合（玩家／電腦）」以及「總手數」。
* **基礎 PVE 模式**：玩家點擊滑鼠翻開棋子後，會切換至電腦回合（電腦目前會自動隨機翻開一顆未知的棋子）。

## 📁 專案結構

確保你的資料夾層級如下，才能正確編譯與執行：

```text
.
├── include/           # Raylib 的標頭檔 (如 raylib.h)
├── lib/               # Raylib 的靜態函式庫 (如 libraylib.a)

├── chinese.ttf        # 遊戲使用的中文黑體字型檔
└── README.md          # 專案說明文件
