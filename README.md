# ♟️ Dark Chess (象棋暗棋) - C 語言與 Raylib 實作

這是一個使用 C 語言與 [Raylib](https://www.raylib.com/) 繪圖函式庫開發的經典「象棋暗棋」遊戲。
專案目前處於開發階段，已實作基礎的棋盤渲染、中文字型顯示、隨機洗牌邏輯，以及基礎的玩家與電腦輪流翻牌功能。

## 🌟 遊戲特色

* **經典 4x8 棋盤配置**：還原傳統暗棋的 32 顆棋子與隨機洗牌機制。
* **完美中文顯示**：透過載入 `.ttf` 字型檔，解決 C 語言繪圖視窗常見的中文亂碼問題。
* **即時資訊面板**：畫面上方會即時顯示「遊戲時間」、「目前回合（玩家／電腦）」以及「總手數」。
* **基礎 PVE 模式**：玩家點擊滑鼠翻開棋子後，會切換至電腦回合（電腦會自動隨機翻開一顆未知的棋子）。

## 📥 如何從零開始編譯與執行這款遊戲？

如果你想在自己的電腦上親自編譯並遊玩這款遊戲，請跟著以下詳細步驟操作：

### 步驟一：安裝 C 語言編譯環境 (w64devkit)
本專案的編譯需要 GCC 編譯器。請按照以下步驟安裝免安裝版的 w64devkit：
1. 前往 GitHub 釋出頁面：[w64devkit Releases](https://github.com/skeeto/w64devkit/releases)
2. 找到最新的版本，點擊下載檔名類似 `w64devkit-x.x.x.zip` 的壓縮檔。
3. **【重要】** 將下載下來的壓縮檔解壓縮，並將裡面的 `w64devkit` 資料夾整個移動到你的 **C 槽底層**。
   *(確認路徑為：`C:\w64devkit\bin`，這會影響後續的編譯指令！)*

### 步驟二：下載本遊戲程式碼
點擊本網頁右上角的綠色 **「<> Code」** 按鈕，選擇 **「Download ZIP」**。下載後，將遊戲專案壓縮檔解壓縮到你的電腦桌面或任何你喜歡的資料夾中。

### 步驟三：快速啟動 (關鍵三步)

準備好環境與程式碼後，接下來請照著做：

#### 1️⃣ 使用什麼開啟？
請使用 **Visual Studio Code (VS Code)**，點擊左上角「檔案 > 開啟資料夾」，選擇你剛剛解壓縮的「遊戲專案資料夾」。接著，在左側檔案列表中點擊打開 **`Dark_Chess.c`** 檔案。

#### 2️⃣ 要開啟哪個視窗？
在 VS Code 上方的選單列選擇「終端機 (Terminal) > 新增終端機」。此時畫面下方會彈出終端機視窗，請**確認目前的路徑顯示的是這個遊戲的資料夾**。

#### 3️⃣ 要輸入什麼指令？
在下方的終端機中，**依序**複製貼上並執行以下三行指令：

**第一行（設定編譯器路徑，請複製並按 Enter）：**
```powershell
$env:Path = "C:\w64devkit\bin;" + $env:Path
```

**第二行（編譯指令，請一字不漏複製並按 Enter）：**
```bash
gcc Dark_Chess.c -o DarkChess.exe -I include/ -L lib/ -lraylib -lopengl32 -lgdi32 -lwinmm -fexec-charset=utf-8 -finput-charset=utf-8
```

**第三行（執行遊戲指令，請複製並按 Enter）：**
```bash
.\DarkChess.exe
```

*(執行後，遊戲視窗就會自動彈出囉！)*

## 📁 專案結構說明

本專案自帶所需的 Raylib 函式庫，無需額外安裝設定：

```text
.
├── include/           # Raylib 的標頭檔 (如 raylib.h)
├── lib/               # Raylib 的靜態函式庫 (如 libraylib.a)
├── Dark_Chess.c       # 遊戲主程式碼
├── chinese.ttf        # 遊戲使用的中文黑體字型檔
└── README.md          # 專案說明文件 (就是你現在看的這份)
```

## 🎮 遊玩操作

* **滑鼠左鍵**：點擊畫面上尚未翻開的灰色暗棋，即可將其翻開並顯示出對應的棋子。
* 玩家操作完畢後，需等待約半秒鐘，電腦會自動執行它的回合。
